# Instruct-FHNet
Road Extraction from Remote Sensing Images with Integration of Building Semantic Information 
从遥感图像中提取道路能够为城市规划与抢险救灾提供数据支持，在城市建设、智能交通以及灾害应急等领域具有重要的应用价值。基于深度学习的遥感图像道路提取方法虽取得一些进展，现有模型仍面临道路错检、断裂以及连通性不足等挑战。鉴于道路与建筑在空间布局上存在的强几何与拓扑关联，本文提出融合建筑语义信息的遥感图像道路提取框架，能够在利用有限标注信息的情况下同时感知道路与建筑物的特征，提升预测道路结果的准确性与连通性。具体而言，首先构建道路-建筑混合指令分割数据集，使模型可根据文本指令动态感知到道路与建筑特征。其次，设计多层级指令语义注入模块，实现视觉特征与任务指令的深度融合；进一步提出频率特征解耦的上下文捕捉模块，分别对图像低频特征与高频特征进行多方向建模，增强模型对遮挡与复杂背景的鲁棒性。最后，采用高斯-卷积混合解码器，协同优化全局语义一致性与局部边界精度，生成兼顾细节与连通性的分割结果。实验表明，所提方法在Massachusetts与Paris数据集上均取得具有竞争力的交并比与路径长度平均相似度。定性实验验证了模型在建筑密集、阴影遮挡等场景的优势，消融实验进一步证实了建筑语义引导与各设计模块的有效性。模型在数据集无法满足建筑物、道路同时标注的情况下仅需一次联合训练即可支持道路与建筑分割。
